# Механизм работы робота
Этот проект посвещён созданию робота, способного убирать мусор из зеленых зон. В данном репозитории будет описана предлагаямая архитектура и дизайн решения.
## Содержание
+ [Цели](#цели)
+ Архитектурные решения
+ Контекстная диаграмма
+ Референсные архитектуры и паттерны
+ Взаимодейтсвие компонентов
+ Дизайн решения
+ Инструкция к запуску
+ Полезные ссылки
+ Рекомендации разработчикам
## Цели
Целью задачи является создание робота который будет убирать мусор из зелёных зон города (парки, заповедники).
